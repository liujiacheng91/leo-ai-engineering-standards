# MSG861 - Msg861X12 结构说明

## 概述

`Msg861X12` 是一个 X12 4010 报文容器对象，用于承载一组 `861` 事务集（`Msg861`）以及报文级的信封段（ISA/GS/GE/IEA）。该类实现 `EDIWritable`，可按指定分隔符将对象序列化写出为标准 X12 文本。

## 类结构图

```text
Msg861X12
├─ isa : Isa
├─ gs : Gs
├─ msg861 : List<Msg861>
├─ ge : Ge
└─ iea : Iea
```

## 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| isa | Isa | ISA | 交换控制头 |
| gs | Gs | GS | 功能组头 |
| msg861 | List\<Msg861> | ST..SE | 861 事务集列表 |
| ge | Ge | GE | 功能组尾 |
| iea | Iea | IEA | 交换控制尾 |

## 输出顺序（write）

`write(Writer, Delimiters)` 的写出顺序与 X12 报文结构一致：

1. 当 `isa` 不为空时输出 `ISA` 段
2. 当 `gs` 不为空时输出 `GS` 段
3. 当 `msg861` 列表非空时，按顺序逐个写出每个 `Msg861`
4. 当 `ge` 不为空时输出 `GE` 段
5. 当 `iea` 不为空时输出 `IEA` 段

其中每个段标识写出后都会追加 `delimiters.getField()` 作为字段分隔符，然后再写出对应对象的字段内容。

## 相关类结构（同目录）

### Msg861

`Msg861` 是单个 861 事务集对象，典型结构为 `ST ... SE`，中间包含基础段、参与方组（N1）以及回执/明细组（RCD）。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| st | St | ST | 事务集头 |
| bra | Bra | BRA | 起始段（语义由标准定义） |
| ref | List\<Ref> | REF | 参考信息列表 |
| dtm | List\<Dtm> | DTM | 时间日期列表 |
| prf | List\<Prf> | PRF | 采购/参考信息列表（语义由标准定义） |
| td5 | Td5 | TD5 | 承运方/路线信息（语义由标准定义） |
| n1Group | N1Group | N1 Group | 参与方组 |
| rcdGroup | List\<RcdGroup> | RCD Group | 回执/明细组列表 |
| se | Se | SE | 事务集尾 |

```text
Msg861
├─ st : St
├─ bra : Bra
├─ ref : List<Ref>
├─ dtm : List<Dtm>
├─ prf : List<Prf>
├─ td5 : Td5
├─ n1Group : N1Group
├─ rcdGroup : List<RcdGroup>
└─ se : Se
```

### RcdGroup

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| rcd | Rcd | RCD | 回执/明细头（语义由标准定义） |
| lin | List\<Lin> | LIN | 行项目标识列表（语义由标准定义） |
| ref | List\<Ref> | REF | 参考信息列表 |
| dtm | List\<Dtm> | DTM | 时间日期列表 |
| td1 | List\<Td1> | TD1 | 运输细节列表（语义由标准定义） |
| slnGroup | SlnGroup | SLN Group | 子行项目组 |

RcdGroup
├─ rcd : Rcd

├─ dtm : List<Dtm>
├─ td1 : List<Td1>
