# MSG301 - Msg301X12 结构说明

## 概述

`Msg301X12` 是一个 X12 4010 报文容器对象，用于承载一组 `301` 事务集（`Msg301`）以及报文级的信封段（ISA/GS/GE/IEA）。该类实现 `EDIWritable`，可按指定分隔符将对象序列化写出为标准 X12 文本。

## 类结构图

```text
Msg301X12
├─ isa : Isa
├─ gs : Gs
├─ msg301 : List<Msg301>
├─ ge : Ge
└─ iea : Iea
```

## 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| isa | Isa | ISA | 交换控制头 |
| gs | Gs | GS | 功能组头 |
| msg301 | List\<Msg301> | ST..SE | 301 事务集列表 |
| ge | Ge | GE | 功能组尾 |
| iea | Iea | IEA | 交换控制尾 |

## 输出顺序（write）

`write(Writer, Delimiters)` 的写出顺序与 X12 报文结构一致：

1. 当 `isa` 不为空时输出 `ISA` 段
2. 当 `gs` 不为空时输出 `GS` 段
3. 当 `msg301` 列表非空时，按顺序逐个写出每个 `Msg301`
4. 当 `ge` 不为空时输出 `GE` 段
5. 当 `iea` 不为空时输出 `IEA` 段

其中每个段标识写出后都会追加 `delimiters.getField()` 作为字段分隔符，然后再写出对应对象的字段内容。

## 相关类结构（同目录）

### Msg301

`Msg301` 是单个 301 事务集对象，典型结构为 `ST ... SE`，中间包含多种业务段与循环组（Y4/N1/R4/LX 等）。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| st | St | ST | 事务集头 |
| b1 | B1 | B1 | 起始段（运输/承运标识等，语义由标准定义） |
| g61 | List\<G61> | G61 | 联系人信息列表 |
| y3 | Y3 | Y3 | 船舶/航次等信息（语义由标准定义） |
| y4Group | List\<Y4Group> | Y4 Group | Y4 组列表 |
| n9 | List\<N9> | N9 | 参考信息列表 |
| n1Group | List\<N1Group> | N1 Loop | 参与方循环组列表 |
| r4Group | List\<R4Group> | R4 Group | 港口/地点组列表 |
| ea | List\<Ea> | EA | 设备/箱信息列表（语义由标准定义） |
| lxGroup | List\<LxGroup> | LX Loop | 明细循环组列表 |
| v1 | List\<V1> | V1 | 船舶信息列表（语义由标准定义） |
| se | Se | SE | 事务集尾 |

```text
Msg301
├─ st : St
├─ b1 : B1
├─ g61 : List<G61>
├─ y3 : Y3
├─ y4Group : List<Y4Group>
├─ n9 : List<N9>
├─ n1Group : List<N1Group>
├─ r4Group : List<R4Group>
├─ ea : List<Ea>
├─ lxGroup : List<LxGroup>
├─ v1 : List<V1>
└─ se : Se
```

### Y4Group

Y4 组对象，由 `Y4` 段与可选的 `W09` 段组成。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| y4 | Y4 | Y4 | 运输相关信息（语义由标准定义） |
| w09 | W09 | W09 | 重量/数量信息（语义由标准定义） |

```text
Y4Group
├─ y4 : Y4
└─ w09 : W09
```

### N1Group

`N1Group` 是 N1 循环组对象，用于承载参与方/地址/联系人等相关段。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| n1 | N1 | N1 | 参与方标识 |
| n3 | List\<N3> | N3 | 地址行列表 |
| n4 | N4 | N4 | 城市/州/邮编/国家 |
| g61 | List\<G61> | G61 | 联系人信息列表 |

```text
N1Group
├─ n1 : N1
├─ n3 : List<N3>
├─ n4 : N4
└─ g61 : List<G61>
```

### LxGroup

`LxGroup` 是 LX 循环组对象，用于承载明细段（N7/K1/L0/L5/L4）以及危险品/处理信息子组（H1/H2）。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| lx | Lx | LX | 明细行号/循环起始 |
| n7 | N7 | N7 | 设备/箱/运输单元信息（语义由标准定义） |
| k1 | List\<K1> | K1 | 备注/说明列表 |
| l0 | L0 | L0 | 数量/重量/体积等（语义由标准定义） |
| l5 | L5 | L5 | 商品/货物描述（语义由标准定义） |
| l4 | L4 | L4 | 特殊处理/装卸信息（语义由标准定义） |
| h1Group | List\<H1Group> | H1 Group | 危险品/处理信息组列表 |

```text
LxGroup
├─ lx : Lx
├─ n7 : N7
├─ k1 : List<K1>
├─ l0 : L0
├─ l5 : L5
├─ l4 : L4
└─ h1Group : List<H1Group>
```
