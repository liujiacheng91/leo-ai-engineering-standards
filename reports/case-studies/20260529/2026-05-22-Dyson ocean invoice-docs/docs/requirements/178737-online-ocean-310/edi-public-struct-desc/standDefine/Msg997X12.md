# MSG997 - Msg997X12 结构说明

## 概述

`Msg997X12` 是一个 X12 4010 报文容器对象，用于承载一组 `997` 事务集（`Msg997`，功能确认/Functional Acknowledgment）以及报文级信封段（ISA/GS/GE/IEA）。该类实现 `EDIWritable`，可按指定分隔符将对象序列化写出为标准 X12 文本。

## 类结构图

```text
Msg997X12
├─ isa : Isa
├─ gs : Gs
├─ msg997 : List<Msg997>
├─ ge : Ge
└─ iea : Iea
```

## 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| isa | Isa | ISA | 交换控制头 |
| gs | Gs | GS | 功能组头 |
| msg997 | List\<Msg997> | ST..SE | 997 事务集列表 |
| ge | Ge | GE | 功能组尾 |
| iea | Iea | IEA | 交换控制尾 |

## 输出顺序（write）

`write(Writer, Delimiters)` 的写出顺序：

1. `isa` → `ISA`
2. `gs` → `GS`
3. `msg997`（按列表顺序逐个写出 `Msg997`）
4. `ge` → `GE`
5. `iea` → `IEA`

其中每个段标识写出后都会追加 `delimiters.getField()` 作为字段分隔符，然后再写出对应对象的字段内容。

### Msg997

单个 997 事务集对象（`ST ... SE`），包含功能组确认段 `AK1/AK9` 与事务集级确认循环组 `AK2/AK3/AK4/AK5`。

| 字段 | 类型 | 段标识 | 说明 |

| ak1 | Ak1 | AK1 | 功能组响应头（语义由标准定义） |
| ak2Group | List\<Ak2Group> | AK2 Group | 事务集响应循环组列表 |
| ak9 | Ak9 | AK9 | 功能组响应尾（语义由标准定义） |
| se | Se | SE | 事务集尾 |

```text
Msg997
├─ st : St
├─ ak1 : Ak1
├─ ak2Group : List<Ak2Group>
├─ ak9 : Ak9
└─ se : Se
```

### Ak2Group

事务集响应循环组：以 `AK2` 起始，包含若干数据段错误循环 `AK3/AK4`，并以 `AK5` 结束该事务集级确认。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| ak2 | Ak2 | AK2 | 事务集响应头（语义由标准定义） |
| ak3Group | List\<Ak3Group> | AK3 Group | 数据段错误循环组列表 |
| ak5 | Ak5 | AK5 | 事务集响应尾（语义由标准定义） |

```text
Ak2Group
├─ ak2 : Ak2
├─ ak3Group : List<Ak3Group>
└─ ak5 : Ak5
```

### Ak3Group

数据段错误循环组：以 `AK3` 描述段级错误，并可包含多个 `AK4` 描述元素级错误。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| ak3 | Ak3 | AK3 | 段级错误信息（语义由标准定义） |
| ak4 | List\<Ak4> | AK4 | 元素级错误信息列表（语义由标准定义） |

```text
Ak3Group
├─ ak3 : Ak3
└─ ak4 : List<Ak4>
```
