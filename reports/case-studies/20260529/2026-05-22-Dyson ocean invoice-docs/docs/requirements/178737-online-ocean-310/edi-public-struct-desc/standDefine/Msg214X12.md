# MSG214 - Msg214X12 结构说明

## 概述

`Msg214X12` 是一个 X12 4010 报文容器对象，用于承载一组 `214` 事务集（`Msg214`）以及报文级的信封段（ISA/GS/GE/IEA）。该类实现 `EDIWritable`，可按指定分隔符将对象序列化写出为标准 X12 文本。

## 类结构图

```text
Msg214X12
├─ isa : Isa
├─ gs : Gs
├─ msg214 : List<Msg214>
├─ ge : Ge
└─ iea : Iea
```

## 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| isa | Isa | ISA | 交换控制头 |
| gs | Gs | GS | 功能组头 |
| msg214 | List\<Msg214> | ST..SE | 214 事务集列表 |
| ge | Ge | GE | 功能组尾 |
| iea | Iea | IEA | 交换控制尾 |

## 输出顺序（write）

`write(Writer, Delimiters)` 的写出顺序与 X12 报文结构一致：

1. 当 `isa` 不为空时输出 `ISA` 段
2. 当 `gs` 不为空时输出 `GS` 段
3. 当 `msg214` 列表非空时，按顺序逐个写出每个 `Msg214`
4. 当 `ge` 不为空时输出 `GE` 段
5. 当 `iea` 不为空时输出 `IEA` 段

其中每个段标识写出后都会追加 `delimiters.getField()` 作为字段分隔符，然后再写出对应对象的字段内容。

## 相关类结构（同目录）

### Msg214

`Msg214` 是单个 214 事务集对象，典型结构为 `ST ... SE`，中间包含业务段与循环组（N1/LX）。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| st | St | ST | 事务集头 |
| b10 | B10 | B10 | 起始段（承运/运输标识等，语义由标准定义） |
| l11 | List\<L11> | L11 | 参考信息列表 |
| man | List\<Man> | MAN | 标记与编号列表 |
| k1 | List\<K1> | K1 | 备注/说明列表 |
| n1Group | List\<N1Group> | N1 Loop | 参与方循环组列表 |
| ms3 | List\<Ms3> | MS3 | 运输方式/承运方信息列表（语义由标准定义） |
| lxGroup | List\<LxGroup> | LX Loop | 明细循环组列表 |
| se | Se | SE | 事务集尾 |

```text
Msg214
├─ st : St
├─ b10 : B10
├─ l11 : List<L11>
├─ man : List<Man>
├─ k1 : List<K1>
├─ n1Group : List<N1Group>
├─ ms3 : List<Ms3>
├─ lxGroup : List<LxGroup>
└─ se : Se
```
