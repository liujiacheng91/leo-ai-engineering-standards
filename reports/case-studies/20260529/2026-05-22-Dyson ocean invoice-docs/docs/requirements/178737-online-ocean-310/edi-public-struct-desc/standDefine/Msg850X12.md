# MSG850 - Msg850X12 结构说明

## 概述

`Msg850X12` 是一个 X12 4010 报文容器对象，用于承载一组 `850` 事务集（`Msg850`）以及报文级的信封段（ISA/GS/GE/IEA）。该类实现 `EDIWritable`，可按指定分隔符将对象序列化写出为标准 X12 文本。

## 类结构图

```text
Msg850X12
├─ isa : Isa
├─ gs : Gs
├─ msg850 : List<Msg850>
├─ ge : Ge
└─ iea : Iea
```

## 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| isa | Isa | ISA | 交换控制头 |
| gs | Gs | GS | 功能组头 |
| msg850 | List\<Msg850> | ST..SE | 850 事务集列表 |
| ge | Ge | GE | 功能组尾 |
| iea | Iea | IEA | 交换控制尾 |

## 输出顺序（write）

`write(Writer, Delimiters)` 的写出顺序与 X12 报文结构一致：

1. 当 `isa` 不为空时输出 `ISA` 段
2. 当 `gs` 不为空时输出 `GS` 段
3. 当 `msg850` 列表非空时，按顺序逐个写出每个 `Msg850`
4. 当 `ge` 不为空时输出 `GE` 段
5. 当 `iea` 不为空时输出 `IEA` 段

其中每个段标识写出后都会追加 `delimiters.getField()` 作为字段分隔符，然后再写出对应对象的字段内容。
