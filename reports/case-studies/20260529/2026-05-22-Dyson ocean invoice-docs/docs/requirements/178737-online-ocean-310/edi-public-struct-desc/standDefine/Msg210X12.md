# MSG210 - Msg210X12 说明文档

## 概述

`Msg210X12` 是一个 X12 4010 报文容器对象，用于承载一组 `210` 事务集（`Msg210`）以及报文级的信封段（ISA/GS/GE/IEA）。该类实现 `EDIWritable`，可按指定分隔符将对象序列化写出为标准 X12 文本。

## 类结构图

```text
Msg210X12
├─ isa : Isa
├─ gs : Gs
├─ msg210 : List<Msg210>
├─ ge : Ge
└─ iea : Iea
```

## 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| isa | Isa | ISA | 交换控制头 |
| gs | Gs | GS | 功能组头 |
| msg210 | List\<Msg210> | ST..SE | 210 事务集列表 |
| ge | Ge | GE | 功能组尾 |
| iea | Iea | IEA | 交换控制尾 |

## 输出顺序（write）

`write(Writer, Delimiters)` 的写出顺序与 X12 报文结构一致：

1. 当 `isa` 不为空时输出 `ISA` 段
2. 当 `gs` 不为空时输出 `GS` 段
3. 当 `msg210` 列表非空时，按顺序逐个写出每个 `Msg210`
4. 当 `ge` 不为空时输出 `GE` 段
5. 当 `iea` 不为空时输出 `IEA` 段

其中每个段标识（如 `ISA`、`GS`、`GE`、`IEA`）写出后都会追加 `delimiters.getField()` 作为字段分隔符，然后再写出对应对象的字段内容。

## 相关类结构（同目录）

### Msg210

`Msg210` 是单个 210 事务集对象，典型结构为 `ST ... SE`，中间包含一组业务段与循环组（N1/LX）。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| st | St | ST | 事务集头 |
| b3 | B3 | B3 | 起始段（运费发票/承运信息等，语义由标准定义） |
| c3 | C3 | C3 | 货币信息（语义由标准定义） |
| b2a | B2a | B2A | 运输方式/装载信息（语义由标准定义） |
| l11 | List\<L11> | L11 | 业务标识/参考信息列表 |
| ref | List\<Ref> | REF | 参考号列表 |
| dtm | List\<Dtm> | DTM | 时间日期列表 |
| n1Group | List\<N1Group> | N1 Loop | 参与方循环组列表 |
| lxGroup | List\<LxGroup> | LX Loop | 明细循环组列表 |
| l3 | L3 | L3 | 汇总/计费信息（语义由标准定义） |
| se | Se | SE | 事务集尾 |

```text
Msg210
├─ st : St
├─ b3 : B3
├─ c3 : C3
├─ b2a : B2a
├─ l11 : List<L11>
├─ ref : List<Ref>
├─ dtm : List<Dtm>
├─ n1Group : List<N1Group>
├─ lxGroup : List<LxGroup>
├─ l3 : L3
└─ se : Se
```

### N1Group

`N1Group` 是 N1 循环组对象，用于承载参与方/地址/联系人等相关段。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| n1 | N1 | N1 | 参与方标识 |
| n2 | N2 | N2 | 参与方补充名称 |
| n3 | List\<N3> | N3 | 地址行列表 |
| n4 | N4 | N4 | 城市/州/邮编/国家 |
| g61 | G61 | G61 | 联系人信息 |
| g62 | G62 | G62 | 日期/时间（参与方相关） |
| l11 | List\<L11> | L11 | 参与方相关的参考信息列表 |

```text
N1Group
├─ n1 : N1
├─ n2 : N2
├─ n3 : List<N3>
├─ n4 : N4
├─ g61 : G61
├─ g62 : G62
└─ l11 : List<L11>
```

### LxGroup

`LxGroup` 是 LX 循环组对象，用于承载明细/货品相关段及其内部循环。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| lx | Lx | LX | 明细行号/循环起始 |
| n9 | List\<N9> | N9 | 参考信息列表 |
| l5 | List\<L5> | L5 | 商品/货物描述列表 |
| l0 | List\<L0> | L0 | 数量/重量/体积等（语义由标准定义） |
| l1 | List\<L1> | L1 | 费率/费用明细列表（语义由标准定义） |
| pod | Pod | POD | 签收/交付信息（语义由标准定义） |
| n1Group | List\<N1Group> | N1 Loop | 明细层参与方循环组列表 |
| g62 | List\<G62> | G62 | 日期/时间列表（明细层） |
| l3 | L3 | L3 | 明细层汇总/计费信息（语义由标准定义） |

```text
LxGroup
├─ lx : Lx
├─ n9 : List<N9>
├─ l5 : List<L5>
├─ l0 : List<L0>
├─ l1 : List<L1>
```
