# MSG315 - Msg315X12 结构说明

## 概述

`Msg315X12` 是一个 X12 4010 报文容器对象，用于承载一组 `315` 事务集（`Msg315`）以及报文级的信封段（ISA/GS/GE/IEA）。该类实现 `EDIWritable`，可按指定分隔符将对象序列化写出为标准 X12 文本。

## 类结构图

```text
Msg315X12
├─ isa : Isa
├─ gs : Gs
├─ msg315 : List<Msg315>
├─ ge : Ge
└─ iea : Iea
```

## 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| isa | Isa | ISA | 交换控制头 |
| gs | Gs | GS | 功能组头 |
| msg315 | List\<Msg315> | ST..SE | 315 事务集列表 |
| ge | Ge | GE | 功能组尾 |
| iea | Iea | IEA | 交换控制尾 |

## 输出顺序（write）

`write(Writer, Delimiters)` 的写出顺序与 X12 报文结构一致：

1. 当 `isa` 不为空时输出 `ISA` 段
2. 当 `gs` 不为空时输出 `GS` 段
3. 当 `msg315` 列表非空时，按顺序逐个写出每个 `Msg315`
4. 当 `ge` 不为空时输出 `GE` 段
5. 当 `iea` 不为空时输出 `IEA` 段

其中每个段标识写出后都会追加 `delimiters.getField()` 作为字段分隔符，然后再写出对应对象的字段内容。

## 相关类结构（同目录）

### Msg315

`Msg315` 是单个 315 事务集对象，典型结构为 `ST ... SE`，中间包含基础段与地点循环组（R4）。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| st | St | ST | 事务集头 |
| b4 | B4 | B4 | 起始段（状态/地点/时间等，语义由标准定义） |
| n9 | List\<N9> | N9 | 参考信息列表 |
| q2 | Q2 | Q2 | 船舶信息（语义由标准定义） |
| sg | List\<SG> | SG | 船舶/航次补充信息列表（语义由标准定义） |
| r4Group | List\<R4Group> | R4 Group | 港口/地点组列表 |
| se | Se | SE | 事务集尾 |

```text
Msg315
├─ st : St
├─ b4 : B4
├─ n9 : List<N9>
├─ q2 : Q2
├─ sg : List<SG>
├─ r4Group : List<R4Group>
└─ se : Se
```

### R4Group

R4 港口/地点组对象，由 `R4` 段起始，后续可包含多个 `DTM` 时间日期段。

#### 字段说明

| 字段 | 类型 | 段标识 | 说明 |
|---|---|---|---|
| r4 | R4 | R4 | 港口/地点信息（语义由标准定义） |
| dtm | List\<Dtm> | DTM | 时间日期列表 |

```text
R4Group
├─ r4 : R4
└─ dtm : List<Dtm>
```
