# MSG304 - Msg304X12 结构说明（按最新映射）

## 概述

`Msg304X12` 是一个 X12 4010 报文容器对象，用于承载一组 `304` 事务集（`Msg304`）以及报文级的信封段（ISA/GS/GE/IEA）。该类实现 `EDIWritable`，可按指定分隔符将对象序列化写出为标准 X12 文本。

本文档以“Java 模型 + 映射文件中定义的出现次数（minOccurs/maxOccurs）”为准更新。

## 类结构图

```text
Msg304X12
├─ isa : Isa
├─ gs : Gs
├─ msg304 : List<Msg304>
├─ ge : Ge
└─ iea : Iea
```

## 字段说明

### Msg304X12

| 字段 | 类型 | 段标识 | 出现次数 | 说明 |
|---|---|---|---|---|
| isa | Isa | ISA | 1..1 | 交换控制头 |
| gs | Gs | GS | 1..1 | 功能组头 |
| msg304 | List\<Msg304> | ST..SE | 1..999 | 304 事务集列表 |
| ge | Ge | GE | 1..1 | 功能组尾 |
| iea | Iea | IEA | 1..1 | 交换控制尾 |

## 输出顺序（write）

### Msg304X12.write

```text
ISA
GS
MSG304*   (每个 Msg304 自身会写 ST...SE)
GE
IEA
```

## 相关类结构（同目录）

### Msg304

`Msg304` 是单个 304 事务集对象，典型结构为 `ST ... SE`，中间包含多种业务段与循环组（N1/R4/LX/L3 等）。

#### 字段说明

| 字段 | 类型 | 段标识/Loop | 出现次数 | 说明 |
|---|---|---|---|---|
| st | St | ST | 1..1 | 事务集头 |
| b2 | B2 | B2 | 1..1 | 起始段（语义由标准定义） |
| b2a | B2a | B2A | 0..1 | 集合用途/目的（语义由标准定义） |
| g1 | G1 | G1 | 0..1 | 装运类型信息（语义由标准定义） |
| n9 | List\<N9> | N9 | 0..200 | 参考信息列表 |
| v1 | List\<V1> | V1 | 0..2 | 船舶信息列表（语义由标准定义） |
| cur | Cur | CUR | 0..1 | 币种信息（语义由标准定义） |
| m2 | M2 | M2 | 0..1 | 销售/交付条款（语义由标准定义） |
| itd | Itd | ITD | 0..1 | 条款/折扣（语义由标准定义） |
| dtm | List\<Dtm> | DTM | 0..20 | 时间日期列表 |
| n1Group | List\<N1Group> | N1-GROUP | 0..100 | 参与方循环组列表 |
| r4Group | List\<R4Group> | R4-GROUP | 0..20 | 港口/地点组列表 |
| k1 | List\<K1> | K1 | 0..12 | 备注/说明列表 |
| l11 | List\<L11> | L11 | 0..99 | 业务参考信息列表 |
| lxGroup | List\<LxGroup> | LX-GROUP | 0..999 | 明细循环组列表 |
| l3Group | L3Group | L3-GROUP | 0..1 | 汇总/计费组 |
| se | Se | SE | 1..1 | 事务集尾 |
