# Custom Skill Template

## Goal

Use this as a starting point for custom LL-compatible Skills.

## Required Sections

1. Goal
2. When to use
3. Inputs
4. Steps
5. Output format
6. Stop conditions
7. References

## Required Compliance

Custom Skills must follow CLAUDE.md and must not bypass risk, naming, output path, verification or security rules.
